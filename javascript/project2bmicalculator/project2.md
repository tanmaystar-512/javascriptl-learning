# ⚖️ BMI Calculator

A clean and functional BMI Calculator. It calculates Body Mass Index based on user input and categorizes the health status (Underweight, Normal, or Overweight).

## 🚀 Features
- **Real-time Calculation**: Instant BMI results based on height and weight.
- **Form Validation**: Checks for invalid inputs using `isNaN()` and logical operators.
- **Dynamic UI**: Uses `innerHTML` to display results and health categories dynamically.

## 🛠 Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (Logic & DOM)**

## 💡 Key Learnings
- Form submission handling with `e.preventDefault()` to prevent page refreshes.
- Data validation using `parseInt()` and `isNaN()` to ensure robust code.
- Implemented conditional rendering for health categories based on calculated BMI.
