# 🎨 Color Switcher Game

A dynamic web application that allows users to change the webpage background color interactively. This project focuses on DOM manipulation and event handling.

## 🚀 Features
- **Interactive UI**: Change background colors with a single click.
- **Dynamic DOM**: Seamlessly updates the `body` style properties.

## 🛠 Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

## 💡 Key Learnings
- Learned to select multiple DOM elements using `document.querySelectorAll()`.
- Implemented `addEventListener()` to handle user interactions effectively.
- Mastered dynamic styling via `style.backgroundColor`.
